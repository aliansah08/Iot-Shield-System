# IoT Shield - Smart Security Monitoring System

## Deskripsi
IoT Shield adalah sistem keamanan untuk perangkat IoT yang mampu mendeteksi serangan seperti brute force dan aktivitas mencurigakan menggunakan Wazuh.

## Tujuan
- Melindungi perangkat IoT dari serangan
- Monitoring real-time
- Mendeteksi brute force SSH

## Arsitektur
- IoT Gateway (Ubuntu Server)
- Wazuh Server (SIEM)
- Agent Monitoring

## Tools
- Wazuh
- Ubuntu Server
- VirtualBox

## Fitur
- Deteksi SSH brute force
- Active response (block IP attacker)
- Monitoring log real-time
